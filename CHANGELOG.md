# Changelog

All notable changes to InspoSearch are documented here.

## [1.1.0] — 2026-04

- Improved query relevance filtering across source categories
- Enhanced service worker resilience and error handling
- Expanded space/astronomy search term coverage
- Improved initial page load performance
- Refined source selection for search relevance

## [1.0.0] — 2026-03

- 196+ sources across museums, archives, nature, science, and photography
- Exact and Explore search modes with intelligent source routing
- 101 languages (6 full UI translations + 95 content languages)
- Europeana integration with 20 sub-collections and dual attribution
- AI vision features (Gemini, Claude, GPT-4o, Ollama)
- Board view for collecting and exporting curated selections
- PWA support with offline caching
- Browser extensions for Chrome and Firefox
- Advanced search with date range, medium, color, and license filters
- Institutions page with partner showcases
