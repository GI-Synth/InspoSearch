InspoSearch API

Free, public REST API for cultural heritage image search.

v1.0 REST JSON

Base URL

https://insposearch-api.<your-subdomain>.workers.dev

Endpoints

GET /search

Search across cultural heritage sources.

GET /search?q=landscape&limit=12

Parameters:

q      (required)  Search query
limit  (optional)  1-100, default 24

GET /sources

List all available sources with metadata.

GET /random

Get random items from a random category.

GET /random?count=6

GET /health

Health check. Returns { "status": "ok" }.

Rate Limits

60 requests per minute per IP address.

Response Format

{
  "query": "landscape",
  "count": 24,
  "results": [
    {
      "id": "artic_12345",
      "title": "A Sunday on La Grande Jatte",
      "artist": "Georges Seurat",
      "date": "1884-1886",
      "thumbnail": "https://...",
      "source": "Art Institute of Chicago",
      "sourceUrl": "https://..."
    }
  ]
}

CORS

Cross-origin requests are allowed from insposearch.pages.dev.
