How to add your collection

Check your API
InspoSearch works with REST APIs, IIIF manifests, OAI-PMH endpoints, and OpenSearch. If your collection has a public API, it can likely be added.
Create a source manifest
Create a JSON file describing your endpoint. See the source templates on GitHub.
Submit a Pull Request
Open a PR on GitHub with your source JSON. We'll test it and add it to the manifest.

Embed widget

Add an InspoSearch search box to your institution's website:

<iframe src="https://insposearch.pages.dev"
  width="100%" height="600"
  style="border:1px solid #eee;"
  title="InspoSearch"></iframe>
