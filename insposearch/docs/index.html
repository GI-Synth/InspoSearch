ROADMAP

Unified Search

Query 2,400+ museums, archives, libraries, and open image sources in parallel. Results stream in as each source responds — no waiting for the slowest link.

Exact + Explore

Exact mode matches your terms precisely for known subjects. Explore mode casts wider across sources for visual discovery. Switch with `m`.

Infinite Scroll That Actually Works

3-lane load-more dispatcher (fresh pages, query variation, pool widening) keeps distinct results coming past page 5. RRF + MMR reduce near-duplicates.

Deep Zoom

IIIF-compatible works open in OpenSeadragon. Examine high-resolution pieces down to brushstroke level — pan and zoom with mouse or keyboard.

Colour Extraction

Dominant colours are extracted from every result. Filter and sort the entire grid by palette — find works by colour relationship, not just subject.

3D Constellation

Visualise results as a navigable point cloud arranged by visual similarity. Surfaces unexpected connections between works.

AI Analysis — No Keys Required

Default analysis runs on Cloudflare Workers AI (LLaVA 1.5). Bring your own key for Gemini, Claude, OpenAI, or Ollama if you want to override.

Cross-Reference (Interpret)

Select any image and press `i`. InspoSearch cross-references it across every source — related works, provenance, and surrounding context in one pass.

Boards

Pin images to a freeform canvas. Arrange, group, compare side by side. Persisted locally; optionally shared via 30-day links.

Keyboard-First

Every action has a shortcut. `/` focus, `m` mode, `i` interpret, `b` board, `s` sources, `k` keys. Built for fast, fluid research workflows.

101 Languages

6 hand-translated base locales plus 95 generated. Locale persisted locally; the app rebinds text on switch without a reload.

Open Source, Zero Lock-In

AGPL-3.0. Vanilla JS. No framework, no runtime dependencies, no tracking. Add sources via JSON manifest + PR. Self-host in one static folder.
