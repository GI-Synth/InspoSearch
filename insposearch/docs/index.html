Unified Search

Query 500+ museums, archives, libraries, and open image sources simultaneously. Results stream in as each source responds.

Exact + Explore Modes

Exact mode matches your terms precisely. Explore mode casts a wider net across all sources for visual discovery.

Board View

Pin images to a freeform board. Arrange, group, and compare results from different sources side by side.

Deep Zoom

IIIF and OpenSeadragon integration for examining high-resolution images down to brushstroke level.

Colour Extraction

Auto-extract dominant colours from any result. Filter and sort by palette across the entire result set.

3D Constellation

Visualise search results as a navigable three-dimensional point cloud based on visual similarity.

AI Research Assistant

Optional integration with Gemini, Claude, OpenAI, or Ollama. Bring your own key. Works fully without AI.

Cross-Reference

Interpret: select any image and cross-reference it against all sources to find related works, provenance, and context.

Keyboard-Driven

Full keyboard navigation. No mouse required. Designed for fast, fluid visual research workflows.

Source Filtering

Toggle individual sources or entire categories. 8 categories including museums, photography, nature, historical, art and design.

Offline Caching

Previously loaded images and metadata are cached locally. Continue browsing your recent searches without a connection.

Open Source

AGPL-3.0 licensed. Add sources via JSON manifest and PR. Community-driven, no vendor lock-in.
