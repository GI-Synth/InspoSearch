Roadmap ​

Planned features and improvements for InspoSearch, roughly ordered by priority.

In Progress ​

  • Documentation site — this site
  • Source health dashboard — live status page showing which APIs are up/down

Planned — Near Term ​

  • Save collections — bookmark images to named collections stored in localStorage
  • Download queue — batch download selected images with attribution metadata
  • Search history — recent queries stored locally with one-click re-search
  • Advanced filters — date range, medium type, source category toggles in search bar
  • Localisation — interface translations starting with FR, DE, ES, NL, JA

Planned — Medium Term ​

  • Similar image search — reverse image search using perceptual hashing
  • Shareable links — URL-encoded search state for sharing queries and results
  • Browser extension — right-click "Search InspoSearch" on any web image
  • PWA — installable progressive web app with offline search history
  • Source plugins — user-submitted source manifests loadable from URL/JSON

Exploring ​

  • Collaborative collections — shared collections via CRDTs (no server)
  • Public API — JSON API for programmatic access to InspoSearch queries
  • Embed widget — embeddable search widget for museum/archive websites
  • Mobile app — native wrapper with offline caching

Completed ​

  • AI research assistant with Gemini/Claude/OpenAI/Ollama (v1.5.0)
  • Interpret: cross-reference (v1.5.0)
  • 3D constellation view (v1.5.0)
  • Board view (v1.4.0)
  • Colour extraction (v1.4.0)
  • Deep zoom via IIIF/OpenSeadragon (v1.4.0)
  • Exact/explore search modes (v1.3.0)
  • Detail view with keyboard navigation (v1.1.0)
  • Reactive state management (v1.2.0)
  • Rate limiter for source manifests (v1.3.0)
  • 500+ sources (v1.5.0)
